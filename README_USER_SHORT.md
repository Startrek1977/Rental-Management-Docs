✨ **One Dashboard** — See everything at a glance: upcoming payments, expiring contracts, renewals, alerts. Your entire portfolio on one screen.

## 🎯 Core Features

**📊 Dashboard** — Real-time snapshot of branches, contracts, monthly rent, renewals, next 5 payments, and expiring contracts.

**🏘️ Branches** — Add unlimited locations with address, trade license, GPS. Attach contacts. Track license expiries.

**📜 Contracts** — Enter annual rent and payment frequency. System generates complete schedule. Set free rent and deposits. Renew or cancel.

**💳 Payments** — Mark paid with date and method (bank/cheque/cash). System calculates outstanding balance.

**📄 Documents** — Upload lease agreements, licenses, contracts. Searchable. One-click access.

**📈 Reports, 🔍 Logs, 🔔 Alerts, 🗺️ Map** — Excel exports, complete audit trail, automatic notifications, interactive map.
